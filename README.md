#JOB PORTAL

```
A Job Searching web application by laravel and rest api with mysql.

JOb seeker can search job post by category or by indivisual name

Job seeker can apply for job , create profile which will search by job poster

JOb poster can post job and search candidate by user experience and job stack



There will be an admin panel for Job seeker and Job poster as well.

A super admin will create job category allow job poster to add jobs and maitain web app security




It will be implemented as a REST API Service and consumed by React application

```

#Create controller with model

```
php artisan make:controller API/v1/Category/CategoriesController --api --model=Model\\Category
```
